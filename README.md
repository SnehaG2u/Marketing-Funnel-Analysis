
# 📊 Marketing Funnel & Conversion Analysis

This project analyzes user behavior across different stages of an e-commerce funnel using Power BI.

## 🔍 Objective
To identify drop-off points and improve conversion rates from View → Cart → Purchase.

## 📁 Dataset
- E-commerce user activity dataset
- Includes event type, product details, user sessions, and pricing

## 📊 Dashboard Features
- Funnel analysis (View → Cart → Purchase)
- KPI cards (Total Views, Cart Adds, Purchases, Conversion Rate)
- Category-wise purchase analysis
- Price range distribution
- User segmentation (New vs Returning)

## 💡 Key Insights
- High traffic but low View → Cart conversion
- Major drop-off at initial stage
- Strong purchase intent after cart stage
- Few categories dominate purchases

## 🛠 Tools Used
- Power BI
- Python (Pandas)
